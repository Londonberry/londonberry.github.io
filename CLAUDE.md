# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Londonberry company website — an Astro + Tailwind CSS static site deployed to GitHub Pages at londonberry.com. The company offers AI-powered custom software development, rapid prototyping, AI strategy consulting, and product management, with a focus on sports technology.

## Development Commands

```bash
npm install              # Install dependencies
npm run dev              # Start dev server (hot reload)
npm run build            # Production build to dist/
npm run preview          # Preview production build locally
```

## Architecture

- **Astro 6 + Tailwind CSS v4** — static site, no SSR
- **Deployment**: GitHub Actions (`.github/workflows/deploy.yml`) builds and deploys to GitHub Pages on push to `main`
- **Site config**: `astro.config.mjs` — site URL, sitemap integration, Tailwind via Vite plugin
- **Tailwind config**: CSS-based via `src/styles/global.css` using `@theme` directives (not a JS config file)
- **Brand colors**: primary `#ff4746`, heading `#487d76`, body `#4b4452`, accent `#e8da5e`, muted `#c9c7cb`, dark `#5d5764`
- **Layout**: `src/layouts/BaseLayout.astro` — shared HTML head, nav, footer, SEO meta, JSON-LD structured data
- **Pages**: `src/pages/` — index, services, work, about, blog, contact
- **Static assets**: `public/` — images, robots.txt, CNAME
- **Design source files**: `design/` directory (git-ignored) contains original 2012 brand assets, PSDs, AI files, fonts (Proxima Nova), and brand guide PDF

## Brand Guidelines

- Logo is a speech bubble icon (diamond point up, three square dots inside) with "LONDONBERRY" wordmark
- Never approximate or hand-draw the logo — always use actual asset files from `design/` or exported versions in `public/images/`
- Font: Proxima Nova (brand), Inter (web fallback via Google Fonts)
- Section labels on white backgrounds use `text-heading/60` (muted teal), not `text-accent` (yellow is hard to read on white)
- Yellow accent color works on dark/teal backgrounds only (footer section headers, sports tech callout)

## SEO/AEO

- Every page has unique meta description, OpenGraph, and Twitter card tags via BaseLayout props
- JSON-LD structured data (ProfessionalService schema) in BaseLayout
- Sitemap auto-generated by @astrojs/sitemap
- robots.txt in public/
