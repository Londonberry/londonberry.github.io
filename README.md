# londonberry.com

Company website for Londonberry — AI-powered custom software development and sports technology, based in Jacksonville, FL.

Built with [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com), deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deployed via GitHub Actions on push to `main`.
